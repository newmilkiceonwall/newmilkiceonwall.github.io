LinuxMint下的GIMP本地化

五月 11, 2023

转载自wade-chiang

关键指令： Link to heading 

sudo apt-get install language-pack-gnome-zh-hans # 这同时也会安装language-pack-gnome-zh-hans-base

GIMP的本地化语言包在这里里面，这个方案也适用于Ubuntu。
