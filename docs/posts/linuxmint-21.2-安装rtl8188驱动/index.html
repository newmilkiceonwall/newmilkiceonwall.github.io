LinuxMint 21.2 如何安装 Realtek RTL8188GU USB 网卡驱动

October 25, 2023 阅读时间：1 分钟

写在前面 链接到标题

这篇文章适用于Realtek Semiconductor Corp. RTL8188GU 802.11n WLAN Adapter 在Linux Mint 21.2 上的安装，理论上，它也适合 Ubuntu 22.04。

步骤 链接到标题

以下的内容参考 TutorialforLinux

  1. 新增PPA
sudo add-apt-repository ppa:kelebek333/kablosuz
sudo apt update
  1. 安装驱动
sudo apt install rtl8188gu-dkms
  1. 加载驱动（重启亦可）
sudo modprobe 8188gu

然后就开可以在托盘栏的网络图标中看到新的无线网卡了。

题外话 链接到标题

配合 Github-linux-wifi-hotspot 可以架设无线热点。
