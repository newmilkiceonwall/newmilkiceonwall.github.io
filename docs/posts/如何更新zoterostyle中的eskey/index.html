我把好一些琐碎问题的解决方案记录在这里。

如何更新zoterostyle中的eskey 链接到标题

使用zotero的时候，插件zoterostyle中影响因子拉不下来。

打开zotero， 编辑——首选项——高级——编辑器——搜索“easys”——用easyscholar的key更新这个字符串

修改Cinnamon鼠标拖动窗口的快捷键 链接到标题

参考这篇文章

在cinnamon的Linux下面使用如下命令修改

gsettings set org.cinnamon.desktop.wm.preferences mouse-button-modifier '<Super>'

Linux Mint 登陆页面启用小键盘 链接到标题 

sudo apt-get install numlockx

系统设置——登陆窗口——设置——勾选“激活numlock”
