Lyx如何修复“未找到引用引擎basic，请安装它。如果你已经安装，请重新配置lyx”问题

January 10, 2024 阅读时间：1 分钟

问题描述 链接到标题

新安装 Lyx，启动时总是提示“未找到引用引擎 basic，请安装它。如果你已经安装，请重新配置 lyx”，多次“重配置”也无效果。根据 [2] ,还可能报错“大部分文档类不存在”。

修复方法 链接到标题

参考 [1] [2] ,在Lyx的安装目录下，找到Lyx\Python\Lib\subprocess.py,这里使用了 Python2.7，在文件前部添加如下代码，修改其使用 UTF-8 编码：

import sys
reload(sys)
sys.setdefaultencoding("utf-8")

启动 Lyx，再次运行“重配置”。
